# Репозиторий лабораторных работ по курсу "Основы алгоритмизации и программирования"

Этот репозиторий содержит коллекцию из 16 лабораторных работ, выполненных в рамках университетского курса по программированию (ОАиП). Работы охватывают два семестра: от изучения основ C++ до разработки полноценных оконных приложений с использованием объектно-ориентированного подхода.

## Цели и задачи курса

Курс был нацелен на получение практических навыков в области:

*   **Язык C++:** От базового синтаксиса до сложных концепций: шаблоны, умные указатели, STL, многопоточность (в санитайзерах).
*   **Объектно-ориентированное программирование (ООП):** изучение классов, наследования, полиморфизма, инкапсуляции, виртуальных функций и таблиц виртуальных методов (vtable).
*   **Структуры данных и алгоритмы:** Реализация динамических массивов (аналог `std::vector`), связных списков, стеков, очередей, бинарных деревьев поиска (BST), куч (heap) и т.д.

**В ходе выполнения лабораторных работ также были получены и закреплены знания по следующим технологиям и инструментам:**

*   **Графические интерфейсы:** Qt
*   **Базы данных и поиск:** PostgreSQL, Elasticsearch
*   **Системы контроля версий:** Git
*   **Системы сборки:** CMake, Makefile, qmake
*   **Библиотеки:** Статические (`.lib`/`.a`) и динамические (`.dll`/`.so`)
*   **Тестирование и отладка:** Google Test, санитайзеры (AddressSanitizer, LeakSanitizer, ThreadSanitizer, UndefinedBehaviorSanitizer), линтеры (C/C++)
*   **Операционные системы:** Linux (Ubuntu), Windows

### 1. Табулирование функций
Освоение среды разработки Linux, компилятора C++20/23, санитайзеров. Разработка алгоритма табулирования функции и изучение основ отладки.

### 2. Ветвления и базовые вычисления
Работа с условными операторами, тернарной операцией, `switch`. Знакомство с Git (создание веток, переключение). Решение вычислительных задач.

### 3. Итерационные циклы
Циклы `while`, `do-while`, `for`. Суммирование рядов с заданной точностью, разложение функций в ряд Тейлора, численные методы поиска корней.

### 4. Массивы
Статические и динамические массивы, ручное управление памятью. Преобразование массивов, обработка матриц, поиск седловых точек, создание "Сапера" и магических квадратов.

### 5. Функции и библиотеки
Рекурсивные функции, раздельная компиляция. Создание статических (`.lib`) и динамических (`.dll`) библиотек.

### 6. Строки и символы
Обработка строк как массивов `char`, написание аналогов строковых функций. Анализ строк, поиск анаграмм и палиндромов.

### 7. Системы счисления и битовые операции
Перевод чисел между системами счисления (прямой, обратный, дополнительный коды). Арифметика в произвольных системах, побитовые операции, решение "задачи о 240 бочках вина".

### 8. Структуры, файлы и базы данных
Работа со структурами (`union` для оптимизации памяти), динамическими массивами структур.
** Выход за рамки курса:**
- **PostgreSQL** — вместо бинарных и текстовых файлов, полноценное реляционное хранилище с CRUD-операциями
- **Elasticsearch** — интеграция поискового движка с поддержкой fuzzy, wildcard и range-поиска
- **Python** — интеграция Python в C++ для автоматической генерации случайных тестовых данных

## 🖼️ Часть 2. Объектно-ориентированное программирование (2-й семестр)

### 1. Оконные приложения. Наследование и полиморфизм
Разработка GUI на **Qt**. Создание иерархий классов "родитель-потомок", виртуальные методы, обработка событий, анимация объектов (автомобили, ракеты, самосвалы и др.).

### 2. Классы, файлы и принципы SOLID
Класс `Date` с методами работы с датами. Оконное приложение для управления данными из файлов (чтение, добавление, изменение). Применение SOLID, обработка исключений. Реализация иерархии классов для арифметических выражений.

### 3. Рекурсия и работа с файловой системой
Рекурсивные алгоритмы (функция Аккермана, Ханойская башня). Оконное приложение для рекурсивного подсчета файлов и папок.

### 4. Продвинутые алгоритмы сортировки и шаблоны
Реализация Heap sort, Quick sort, Merge sort. Шаблонный контейнер `Vector` (аналог `std::vector`) с итераторами и основными методами.

### 5. Динамические библиотеки и работа со строками
Статические и динамические библиотеки. Собственный класс `String` (аналог `std::string`) с реализацией строковых функций (`strcpy`, `strcat`, `strcmp` и др.), "Правило трех".

### 6. Структуры, указатели, ссылки и файловый ввод/вывод
Структуры по варианту. Работа с указателями и ссылками. Операции записи/чтения в текстовые и бинарные файлы (посимвольная, потоковая).

### 7. Динамические структуры данных
Реализация очередей (deque, кольцевые), бинарных деревьев поиска (BST), хеш-таблиц с разрешением коллизий. Все основные операции, обходы, анализ сложности, построение графиков.

### 8. Продвинутые деревья и контейнеры
Шаблонные `Map` и `Set` (аналоги `std::map`/`std::set`) с итераторами. Бинарная куча на массиве и списке. Хеш-таблица с открытой адресацией.
